# 1. Soumission manuelle

Créer une fiche de production directement depuis l'interface.

## Étapes

1. **Cliquez sur l'onglet "Soumission"** dans la barre d'outils principale (visible pour les profils Admin, Opérateur, Soumission).
2. **Glissez-déposez un PDF** dans la zone de dépôt, ou cliquez pour sélectionner un fichier.
3. **Renseignez la fiche** : intitulé, quantité, format, support, finitions, date de livraison, etc.
4. **Cliquez sur "Soumettre"** — la fiche est créée dans le dossier "Début de production" du kanban.

## Conseils

- Le fichier PDF est automatiquement déplacé dans le hotfolder correspondant (`Début de production`).
- Si vous activez le preflight automatique, le PDF est analysé immédiatement après la soumission.
- Vous pouvez joindre **plusieurs PDF** à la même fiche via le bouton "+ Ajouter un fichier".

## Champs obligatoires

- **Intitulé** — Nom du job affiché dans le kanban.
- **Quantité** — Nombre d'exemplaires.
- **Date de réception souhaitée** (si configuré comme obligatoire).
