# GestionAtelier
Gestion Atelier d'impression offset et numérique

## Description

Application web de gestion d'atelier d'impression offset et numérique, développée avec React + Vite.

## Fonctionnalités

- **Tableau de bord** : Vue d'ensemble avec KPIs (travaux en cours, machines, alertes stock, CA)
- **Clients** : Gestion complète des clients (ajout, modification, suppression, recherche)
- **Machines** : Suivi des machines offset et numériques (état, maintenance, caractéristiques)
- **Travaux** : Gestion des travaux d'impression (devis, statut, planning, facturation)
- **Inventaire** : Gestion des stocks (papiers, encres, consommables) avec alertes de seuil
- **Rapports** : Rapports de chiffre d'affaires, charge machines, répartition des travaux
